# Mizhab Mujeeb NP - DevOps & Cloud Engineer Portfolio

A modern, production-grade portfolio website built with Next.js 15, TypeScript, Tailwind CSS, Framer Motion, and Lucide Icons.

## Features

- **Next.js 15 App Router** with static export
- **TypeScript** for type safety
- **Tailwind CSS** with custom AWS-inspired design system
- **Framer Motion** animations and scroll-triggered effects
- **Dark/Light Mode** via next-themes
- **Responsive Design** optimized for all devices
- **SEO Optimized** with OpenGraph and metadata
- **Glassmorphism UI** with AWS color palette
- **Scroll Progress Indicator**
- **Back-to-Top Button**
- **Animated Counters** and **Skill Bars**

## Sections

1. Hero with profile and CTAs
2. About Me with animated skill cards
3. Technical Skills with proficiency indicators
4. Certifications showcase
5. Featured Projects (THROW CI/CD)
6. DevOps Labs grid
7. Project Documentation Hub with search/filter
8. Architecture Diagrams gallery with zoom
9. Experience & Learning Timeline
10. GitHub Activity Dashboard
11. Contact form with social links
12. Professional footer

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

The static export will be generated in the `out/` directory.

## Design Philosophy

Inspired by AWS Console, modern SaaS dashboards, and GitHub aesthetics. This portfolio is designed to feel like a professional engineer's personal cloud platform rather than a standard web developer portfolio.
